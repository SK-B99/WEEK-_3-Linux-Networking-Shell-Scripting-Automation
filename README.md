WEEK 3: DAILY BREAKDOWN
Module: Linux Networking, Shell Scripting & Automation
Theme: "Automate your work; don’t repeat it."

DAY 1 – Networking Fundamentals
Topics: IP addresses, DNS, ping, traceroute, curl, wget
Lab: Use curl to fetch a webpage and save the output to a file
Challenge: Diagnose network reachability using ping + traceroute on any public domain

DAY 2 – Network Tools & Service Diagnostics
Topics: netstat / ss, nc (netcat), telnet, checking open ports, simple debugging
Lab: Use nc to test connectivity between two ports
Challenge: Identify which service is running on an unknown open port

DAY 3 – Shell Scripting Basics
Topics: Variables, conditionals (if/else), loops, functions, script execution (chmod +x)
Lab: Write a script that accepts a filename and outputs the number of lines, words, and characters
Challenge: Create a script that alerts you if a directory exceeds a specific size

DAY 4 – Automation & Task Scheduling
Topics: cron jobs, logging, script scheduling, automation patterns
Lab: Create a cron job that runs a script every 5 minutes and writes output to a log file
Challenge: Automate a daily backup of a specific directory into a timestamped archive

DAY 5 – Networking + Scripting Final Lab
Topics: Combining CLI networking tools with automation
Demo: Build a simple monitoring script for uptime or endpoint status
Final Lab Task:

Write a Bash script that:
Pings a list of predefined servers
Logs their online/offline status with timestamps
Sends results to a log file
Runs automatically via cron
