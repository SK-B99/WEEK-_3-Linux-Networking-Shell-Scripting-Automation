`cron` is a daemon to execute scheduled commands. In Linux, a daemon is a non-interactive program that operates continuously in the background. A cron job is a Linux program that allows users to schedule the execution of a piece of software, often in the form of a shell script or a compiled executable.

### Common crontab Commands
- List cron jobs: `crontab -l`
- Start the cron service: `sudo systemctl start cron`
- Stop the cron service: `sudo systemctl stop cron`
- Remove all cron jobs: `crontab -r`

Cron jobs typically follow this format: **minute (m)**, **hour (h)**, **day of the month (d)**, **month (m)**, **day of the week (w)**.

### Example of a cron job
*/2 * 2,15 * * echo "hi from cron" >> /home/sammy/ex.txt


This will run the command and append "hi from cron" to the `ex.txt` file **every 2 minutes**, but **only on the 2nd and 15th days** of each month, regardless of the month or day of the week.
